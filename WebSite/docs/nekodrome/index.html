The Architecture

NekoDrome is a bespoke, high-fidelity client built specifically to interface with your Navidrome server. It leverages the PlayStation Vita's unique hardware for a native, lag-free experience.

  • Direct Navidrome Sync: Built exclusively for Navidrome’s API endpoints for maximum library and artwork compatibility.
  • Asynchronous Multi-Threading: UI rendering and music streaming operate on separate threads for a stutter-free 60FPS experience.
  • Optimized Audio Support: Native playback for MP3, FLAC, OGG, and WAV directly through Vita hardware.

Sonic Capabilities

  • Offline "Capture": Save any track or album locally to ux0:data/nekodrome for full playback without Wi-Fi.
  • Intelligent Queueing: Automatically populates smart queues based on your Navidrome album or playlist context.
  • Seamless Transitions: Pre-buffers the next track in the queue to minimize silence between songs.
  • Instant Search: Use the Vita's native keyboard to index and filter through your entire library in milliseconds.

THE INPUTS

D-PAD: Navigate the UI. Up/Down moves between entries. Left/Right skips 50 albums at a time (changes album pages).

CROSS: Action. Enter albums or Toggle Play/Pause.

SQUARE: Capture. Download track to local storage for offline use.

TRIANGLE: Find. Trigger the system keyboard to search.

L/R: Switch. Change the current page.
